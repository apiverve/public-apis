# Contributing to APIVerve APIs

Thank you for your interest in APIVerve APIs!

## Reporting Issues

Found a bug or have a feature request?

1. Check if the issue already exists
2. If not, [create a new issue](../../issues/new)
3. Provide as much detail as possible:
   - API name
   - Expected behavior
   - Actual behavior
   - Steps to reproduce
